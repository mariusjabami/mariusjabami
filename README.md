# Marius Jabami

**ML Engineer | Low-Level Systems Specialist | Founder @ λχ Corp**

Building efficient, from-scratch AI and systems for resource-constrained environments. Passionate about low-level programming, open source, edge computing, and making advanced technology accessible in emerging markets like Angola.

### Featured Projects

- **[ASP – Adaptive System Philosophy](https://github.com/mariusjabami/ASP)**  
  A complete Multi-Layer Perceptron (MLP) neural network implemented 100% in pure C — no dependencies beyond GCC.  
  Features manual backpropagation, Xavier initialization, ReLU/sigmoid/step activations, CSV reader, and CLI tools for training/inference.  
  Proven performance: \~83% accuracy on Pima Indians Diabetes dataset; trains in <1 minute on 1GB RAM Android devices. Ideal for embedded/IoT and low-resource ML.  
  License: MIT | Stars growing

- **[Medusa](https://github.com/mariusjabami/medusa)**  
  High-performance CLI agent for deep code and repository analysis powered by Gemini 3 API.  
  Built for the Google DeepMind Gemini 3 Hackathon. Traverses directories, detects patterns, generates structured reports.  
  Written in pure C with libcurl + cJSON for maximum portability and speed.

- **[synap](https://github.com/mariusjabami/synap)**  
  From-scratch neural network in C++: manual gradient descent, quadratic loss, non-linear activations.  
  A practical study in deep learning fundamentals without frameworks.

- **[j32](https://github.com/mariusjabami/j32)**  
  Bare-metal kernel development for ARM32 architecture — bootloader, memory management, and core drivers from scratch.

- **[λχ Corp Models (Hugging Face)](https://huggingface.co/lxcorp)**  
  Lightweight LLMs optimized for Portuguese and low-resource inference: lambda-1v-1B, Link-270M GGUF, Synap-2b, WNL468M.  
  Focus: education, logical reasoning, and edge deployment.
